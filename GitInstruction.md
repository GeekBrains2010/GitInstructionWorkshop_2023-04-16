## Установка git и VSCode на ваш ПК.
Для начала загружаем на свой ПК программу 
 [Visual Studio Code ](https://code.visualstudio.com) и программу [GIT](https://git-scm.com/downloads)  
Устанавливаем обе программы соблюдая инструкции установщика программ.  
Запускаем программу Visual Studio Code. В открывшемся окне выбираем пункт Открыть папку. Создаем новую папку и выбираем ее. Соглашаетесь с тем, что вы доверяете автору. Создаем новый файл в выбраной папке и присваеваем ему название. Далее вызываем функцию Терминала Файл/Вид/Терминал. В открывшемся окне терминала вводим /git config --global user.name «  » , между скобок вводим имя (латиницей) Далее вводим команду git config --global user.email fjgfkjg@example.com адрес почты необходимо вводить действующий, так как он в дальнейшем адрес почты будет необходим для создания облачного хранилища данных. После ввода имени и почты данная программа запоминает вас как пользователя.
